# Skillr - Competitive Gaming Platform

Skillr is a competitive gaming platform that allows users to register for tournaments, compete, and earn rewards.

## Features

- User Authentication
- Tournament Registration
- Mobile Money Payments
- Real-time Updates via WebSocket
- Tournament Brackets
- Leaderboards
- Social Features (Teams, Friends, Chat)
- Advanced Analytics

## Tech Stack

- **Backend:** Node.js, Express, MongoDB, WebSocket (ws)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MongoDB

## Setup

1.  Install dependencies:
    ```bash
    cd server
    npm install
    ```

2.  Set up environment variables in `server/.env` (optional, defaults provided in code).

3.  Start the server:
    ```bash
    cd server
    npm start
    ```

4.  Open `http://localhost:5000` in your browser.
